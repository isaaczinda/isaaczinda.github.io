Welcome to RainCheck.

RainCheck gets the raw capacitance matrix from the Nexus 5 phone, and uses this data to improve the performance of the phone's touchscreen when wet.

This project was completed by Isaac Zinda with the help of Mayank Goel in Professor Shwetak Patel's UbiComp Lab at University of Washigton.
