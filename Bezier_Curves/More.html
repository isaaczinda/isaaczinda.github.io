Why Is This Function Fun?

The Bezier curve is a fun function because it is used for computer graphics and entertainment. Bezier curves are used in game design to accelerate an object at a varying but realistic speed, and are also used to trace motion paths in games. They can be used to represent terrain in side-scrollers, like Canvas Rider, Tiny Wings, or Happy Wheels. In addition, all graphic design programs use Bezier curves to draw curved shapes, because setting them up is easy and intuitive. These curves are also used in developing the curves in letters for some fonts, notably True Type fonts.
